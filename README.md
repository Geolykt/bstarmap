bStarmap
===

bStarmap is the first true open deobfuscation mappings project for Galimulator.

Now, bStarmap is a bit of a gambit. With b standing for blight, it is expected
to quickly propagate throughout most of the Galimulator modding scene and to
facilitate Galimulator modding for newcomers.

However, it represents a departure of the tried-and-true system of semiautomatic
deobfuscation mappings in the form of spstarmap. On it's own it shouldn't be an
issue, but if Galimulator were to ever get updated, then well that's it.
I honestly don't believe that it would be possible to update bStarmap to
a new Galimulator version so the gambit I'm making here is to assume that
Galimulator is a dead game, which honestly it might not be - at least not yet.

But only time will tell.

As for the license, this project uses CC0. This is the same license as fabricmc's
yarn, which is a deobfuscation project in a similar vein just for Minecraft.
